Musify Applications


DESCRIPTION
------------------------------
The goal of this project is to create an iPhone/iPad Music app using iTunes affiliate API.


BUILD REQUIREMENTS
------------------------------
iOS 12.0 SDK or later



SUPPORTED FEATURES
------------------------------

* Audio Player
* Search Music By Artist Name


REQUIREMENTS TO BUILD THE APPLICATION
------------------------------

* XCode 12 or later
